保險（127 題 · 25.2%）

你的主場 · 預估考 12~13 題 · 目標全拿
💡 你的優勢

保險是最大宗主題。身為南山人壽從業人員，這塊幾乎不用額外準備，快速複習關鍵數字和容易混淆的概念即可。

風險管理基礎（15 題）

風險分類與管理方法
  • 純粹風險 vs 投機風險：保險只承保純粹風險（只有損失可能，無獲利可能）
  • 風險因素 → 風險事故 → 損失（因果關係方向，不可反過來）
  • 基本風險：影響整個群體或大部份人的風險
  • 可保風險要件：損失可預測、明確可衡量、大量「同質性」風險單位、可能造成重大損失
風險管理方法對照
方法適用情境功能
風險避免完全消除風險頻率降為零（唯一能做到的）
損失預防降低損失發生機率例：檢修汽車
損失抑減減少已發生損失的嚴重程度例：搶救車禍傷患
風險自留損失幅度小、成本可估計自己承擔
保險損失頻率小但幅度大轉嫁風險

人身保險（42 題）

壽險分類
險種特色適合對象
定期壽險期滿生存保障全失、保費最低養育子女期間：遞增型
終身壽險保障終身、有儲蓄成分長期保障需求
生存保險目的為儲蓄預備學費、退休金
生死合險（養老）= 生存險 + 定期險工作年齡保障 + 晚年準備
年金保險生存期間定期給付退休生活
投資型保險連結投資帳戶、風險由保戶承擔有投資需求者
投資型保險重點
  • 變額壽險：保費固定、保險金不固定、分離帳戶
  • 萬能變額壽險：現金價值不固定、較低保證利率、保障彈性大
  • 不具保證收益 → 選項出現「保證一定收益」就是錯的
其他人身保險
  • 傷害保險：意外事故 = 外來 + 突發 + 非疾病。保費主要考量因素 = 職業類別（非年齡）
  • 健康保險：包含失能保險（非傷害保險）
  • 團體保險：最低 5 人、不需體檢、一張總保單
  • 死亡給付：被保險人應年滿 15 歲
  • 貸款購屋：配合遞減型定期保險（不是遞增型）

財產保險（32 題）

保險分類
  • 保險法分類：財產保險（火災/陸空/責任/保證/其他）＋人身保險（壽險/健康/傷害/年金）
  • 財產保險不承保：土地（但承保建築物、動產、收益）
  • 財產保險特性：期間短（通常1年）、保險金額差距大、依賴再保險較深
  • 保險利益須在保險事故發生時存在（非要保時）
  • 複保險分攤：保險金額比例分攤法
火災保險
  • 商業火災基本承保：火災、閃電雷擊、爆炸引起之火災、竊盜後縱火
  • 不承保：單純爆炸（要附加）、汽車
  • 保險金額基礎：實際價值（商業火災）/ 重置成本（住宅火災）
  • 共保比例：商業 80% / 住宅 60%
汽車 / 責任 / 強制險
  • 車體損失險：甲式（最廣）→ 乙式 → 丙式（免自負額車對車碰撞）
  • 零件理賠：以新品為準（不減折舊）
  • 責任保險 = 第三人責任保險，承保民事責任
  • 強制車險死亡給付：200 萬元，不含財損給付
  • 住宅地震基本保險賠償限額：150 萬元
  • 強制車險續保通知：期滿前 30 日

社會保險（15 題）

各類社會保險對照
保險主管機關關鍵考點
勞工保險勞動部15~65 歲；職災只保傷病/醫療/失能/死亡（不含生育/失業）
就業保險勞動部僅限本國籍受雇勞工；不含公教、外籍
公教保險銓敘部不含醫療給付
軍人保險國防部僅限現役軍人；不含育嬰留停津貼
農民健康保險內政部不含失業給付、老年給付
全民健保衛福部不含失能；眷屬保費最高3 口
⚠ 高頻陷阱

勞保普通事故＋職災「都有」的給付只有：傷病、失能、死亡。「生育」只在普通事故，「失業」屬就業保險。就業保險只含本國籍，軍人保險只指現役。

高齡與弱勢（7 題）
  • 高齡者：65 歲以上（老年福利法）
  • 弱勢金融消費者：高齡者、身心障礙者、低學歷者（非銀行理專、非專業投資人）
  • 身心障礙者投保：不得歧視，應依障礙種類規劃適當商品
  • 微型保險：提供特定身分及經濟弱勢者的保險商品
  • 金融消費爭議處理機構：金融消費評議中心

銀行（76 題 · 15.1%）

預估考 7~8 題 · 生活常識 + 必記數字
銀行體系架構
  • 商業銀行：一般存放款、匯兌業務（非專業銀行）
  • 專業銀行：工業信用、中小企業、輸出入信用（商業銀行不算）
  • 金融控股公司：結合銀行/證券/保險，母公司不可由非金融產業主導
  • 存款保險：每人每行最高 300 萬元
  • 資本嚴重不足：自有資本比率低於 2%

存放款業務（22 題）

存款種類
  • 支票存款：可簽發支票
  • 活期/活期儲蓄存款：隨時存取
  • 定期存款：有一定時期限制，可質借
  • 活期存款利息：須併入綜所稅申報（非分離課稅）
授信原則與擔保
  • 授信基本原則：安全性、流動性、收益性（不含公開性、投機性）
  • 擔保授信：不動產/動產抵押、權利質權、信保機構保證、營業票據
  • 非擔保授信：借款人開立之票據、鄉鎮公所保證函
  • 已足額擔保的自用住宅貸款：不得再徵提連帶保證人
  • 房屋保險受益人 = 銀行（不是借款人）
  • 利率最低：首次購屋貸款（低於信貸/車貸/現金卡）
信用卡/現金卡
  • 學生信用卡：年滿 18 歲、最多 3 家、每家上限 2 萬
  • 循環信用：繳最低應繳金額，未繳部分按月收息
  • 收單銀行：接受特約商店持簽帳單請款
  • 現金卡：與卡片結合的無擔保消費性貸款
銀行風險管理
風險類型定義
信用風險借款人或交易對手違約
市場風險利率/匯率/股價等市場因素變動（不含特定企業營運）
流動性風險金融資產變現能力不足或無法以合理價格軋平部位
營運風險人為疏失、舞弊、資訊系統問題（內控不完備）
利率風險利率敏感性缺口控制

⚠ 逾放比率越低 = 資產品質越好（反向關係）

內部控制
  • 內控最終責任：董(理)事會（非總經理）
  • 內控措施：日常牽制、命令休假、內部查核（不含獎懲制度）
  • 稽核人員不得兼任任何業務職務

外匯市場（13 題）

外匯基礎
  • 即期交易：簽約後第 2 個營業日（T+2）交割
  • 遠期交易：約定未來某日以約定匯率交割
  • 交叉匯率：透過第三種通貨間接計算
  • 換匯匯率：遠期匯率與即期匯率的差額
  • 報價 EUR/USD 1.1713/17 → 詢價者買入歐元價格 = 1.1717（較高的那個）
  • 本國貨幣貶值因素：進口關稅下降、相對物價上漲
  • 外匯干預：央行調節匯價的手段
  • 外資避險：買 NDF（無本金交割遠期）

證券（160 題 · 31.7%）

最大主題 · 預估考 16 題 · 需要重點突破
📊 證券是金融常識的決勝區

佔比最高但你比較不熟，這裡多拿 5 分就能確保過關。股票和債券的基本概念搞懂就能拿到大半分數。

市場架構（28 題）
  • 集中市場 = 交易所（台灣證券交易所），集中競價撮合
  • 店頭市場（OTC）= 交易商報價，投資人接受即成交
  • 初級市場 = 發行市場（IPO、現金增資）
  • 次級市場 = 流通市場（已發行證券的買賣）
  • 台股交易方式：電腦自動交易（非人工喊價）
  • 結算交割：臺灣證券集中保管結算所
  • 興櫃市場：未上市未上櫃股票合法流通
三種證券商
類型角色主要活躍市場
承銷商幫企業銷售新發行證券發行（初級）市場
經紀商受託執行買賣雙方交易流通（次級）市場
自營商自行買賣有價證券流通（次級）市場

股票市場（60 題）

股票基礎
  • 普通股：有表決權、參與經營、承擔經營成敗（不可優先分配股息）
  • 特別股：固定股利、優先分配、通常無表決權
  • 本益比 = 股價 ÷ 每股盈餘
  • 投資報酬率 = (賣價 - 買價 + 股息) ÷ 買價
交易制度
  • 漲跌幅：開盤競價基準上下 10%
  • 交易方式：集合競價交易制度
  • 零股買賣：999 股以下
  • 鉅額交易：500 單位以上
  • 盤後定價交易：依收盤價
  • 當日沖銷：不收融資利息
  • 交易成本：手續費（買賣雙方）+ 交易稅（賣方）
  • 信用帳戶擔保維持率：每日計算
除權除息計算

除權參考價 = 收盤價 ÷ (1 + 每股配股/面額)

例：收盤價 60 元，每股配 2 元股利 → 60 ÷ (1 + 2/10) = 50 元

融資融券
  • 融資 = 借錢買股（看多），成本含融資利息
  • 融券 = 借股賣出（看空），成本含融券手續費（不含融券利息）
  • 保證金不足 → 補足至原始保證金
承銷方式
方式特色
競價拍賣價格最具市場性
公開申購配售最具公平性及股權分散
詢價圈購洽特定法人
上市上櫃條件
  • 上市：設立登記屆滿 3 年
  • 上櫃：須 2 家以上證券商書面推薦
  • 年度財報：年度終了後 3 個月內公告
  • 股東常會：年度終了後 6 個月內召開
  • 公告場所：公開資訊觀測站

債券市場（28 題）

債券核心概念
  • 利率↑ → 債券價格↓（反向關係）
  • 到期日越長 → 利率風險越大 → 價格敏感性越大
  • 高票面利率債券 → 對殖利率變動的價格敏感性較小
  • 債券到期 → 價格趨近面值
  • 公司債持有人 = 債權人（非股東）
  • 可贖回公司債贖回權 = 發行公司
  • 投資等級：S&P BBB 以上
  • 垃圾債券 = 高風險高殖利率（非低報酬/非投資等級）
債券分類
類型特色
零息債券以低於面額發行、到期按面額贖回、票面價格 > 市價
可轉換公司債可轉為股票、凍結期間越長 → 價值越低
外國債券A國在B國發行以B國幣計價
歐洲債券A國在B國發行不以B國幣計價、條件較彈性
認購權證不屬於債券、發行人 = 綜合證券商

公司債交易稅 = 免稅；存託憑證：赴美 = ADR、來台 = TDR

基金投資（33 題）

基金分類
開放型封閉型
單位數非固定（可申購贖回）固定
交易方式向投信按淨值贖回在集中市場按市價交易
贖回壓力有無
折溢價無有（多數折價）
數量遠多於封閉型較少
基金投資重點
  • 投信 = 發行基金的公司 ≠ 投顧（提供投資建議）
  • 基金經理人不能保證獲利（出現「保證」就是錯的）
  • 定期定額：股價越低買越多單位、股價越高買越少
  • 年輕人 → 積極成長型；老年人 → 保本型/定存
  • 積極式管理 = 選股/擇時；消極式 = 買指數型基金
  • 貨幣市場基金：適合短期資金停泊
  • 共同基金 vs 對沖基金：共同基金追求相對報酬、不積極買賣衍生品
  • 績效評比應用風險調整後報酬率（非絕對報酬率）

其他主題（141 題 · 28%）

期貨選擇權 / 信託 / 貨幣市場 / 央行 / 金融科技 / 洗錢防制
期貨選擇權（30 題）
  • 期貨：標準化合約、每日結算、買賣雙方信用風險由結算所承擔
  • 選擇權買方：有權利無義務、不需繳保證金
  • 選擇權賣方：有義務、需繳保證金
  • Call = 買權（看漲）、Put = 賣權（看跌）
  • 正向市場：期貨價格 > 現貨價格
  • 到期時：期貨價格 = 現貨價格（因為要交割）
  • 股價指數期貨：現金交割（非實物）
  • 保證金不足 → 補至原始保證金
  • 認購權證發行者 = 賣出買權（Sell Call）
  • 停損單：買在市價之上、賣在市價之下
信託（22 題）
  • 三方：委託人（交付）→ 受託人（管理）→ 受益人（享有）
  • 設立方式：契約/遺囑/宣言（不含法定信託）
  • 受託人不得：承諾擔保本金或最低收益率、自有財產與信託財產合併管理
  • 信託財產：不屬於受託人遺產或破產財團
  • 消滅原因：信託目的不能完成（非委託人死亡/受託人破產）
  • 營業信託契約：須以書面為之
  • 公益信託監督機關：目的事業主管機關
  • 特定金錢信託：投資標的由委託人指定（最常見基金投資方式）
  • 信託業不得經營：商譽信託、人格權信託
貨幣市場 + 票券（20 題）
工具說明
國庫券（T-Bill）政府發行、未滿1年、央行經理
商業本票（CP）融資性須經信評機構評等之金融機構保證
可轉讓定存單（NCD）最低1個月、最長1年、面額10萬倍數、可轉讓不可中途解約
銀行承兌匯票（BA）銀行為付款人並承兌
附買回協議（RP）先賣後買回

短期票券利息分離課稅 10%；票券集中保管結算機構限 1 家；款項清算由央行同資系統

央行與貨幣政策（9 題）
  • 央行非隸屬金管會、獨占發行通貨、保管國際準備
  • 不直接對企業融通資金
  • 公開市場操作 = 最常用、最機動的政策工具
  • 寬鬆政策：買進債券/NCD、降低準備率/重貼現率
  • 國際收支順差 → 貨幣供給增加
  • 法定準備率降低 → 存款貨幣創造能力提高
  • 非央行工具：調整稅率（那是財政政策）
洗錢防制 + 舞弊（9 題）
  • 三階段：處置 → 多層化 → 整合
  • 大額通貨：50 萬元以上申報
  • 申報對象：法務部調查局
  • 舞弊三角：壓力/動機 + 機會 + 合理化
  • 內線交易禁止期：重大消息公開後 18 小時內
  • 拒絕開戶情形：影本、拖延補件、匿名假名人頭
金融科技（7 題）
  • 純網路銀行特色：無實體分行/ATM、全年無休（但透過 APP 提供服務不是獨特特色）
  • 台灣核准：連線銀行、將來銀行、樂天銀行（不含「新網銀行」）
  • 比特幣風險：洗錢 + 資恐 + 逃稅 + 詐騙（全部都是）
  • AI = 機器人的大腦；生物辨識 = 網路/電話驗證身分
  • 1TB = 1024GB
  • 「數據資料」≠「大數據」（其他三個同義）
